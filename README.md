# 🚀 INDIAIN

### AI-Based Career Intelligence & Placement Readiness Platform

INDIAIN is an **AI-powered Career Intelligence System** designed to help students and early professionals analyze their **skills, placement readiness, and career opportunities**.

Unlike traditional job portals, INDIAIN focuses on **skill verification, placement preparation, and market intelligence** to help users become industry-ready.

> 🧠 Think of INDIAIN as your **AI Career Advisor + Placement Readiness System**

---

# 🌍 Vision

Most platforms like **LinkedIn** or **Internshala** only show job listings.

INDIAIN goes beyond that by helping users:

✔ Analyze their **skills**
✔ Identify **skill gaps**
✔ Improve **resume quality**
✔ Track **coding progress**
✔ Discover **job opportunities**
✔ Generate **career roadmaps**

The goal is to transform job searching into a **data-driven career development journey.**

---

# 🎯 Core Features

## 1️⃣ Smart Profile System

Users create a detailed professional profile including:

* CGPA
* Skills
* Projects
* Certifications
* GitHub Profile
* LeetCode progress
* Resume upload
* Preferred role (SDE, Data Analyst, Cybersecurity, etc.)

All this data is used by INDIAIN to **analyze career readiness and skill gaps.**

---

# 🧠 Skill Match Engine (Core Brain)

The system compares a user's skills with **industry-required skills for specific roles**.

### Example Output

Role: Backend Developer

Skill Match: 68%

Missing Skills:

* Node.js
* SQL Optimization
* REST API Design

Recommendation:

* Solve 50 more DSA problems
* Build one backend project

### Formula

Skill Match % =
(Common Skills / Required Skills) × 100

---

# 📊 Placement Readiness Score

INDIAIN calculates an overall **Placement Readiness Score out of 100**.

### Score Factors

| Factor          | Weight |
| --------------- | ------ |
| Skills          | 30%    |
| Projects        | 20%    |
| GitHub Activity | 20%    |
| DSA Progress    | 20%    |
| Resume Quality  | 10%    |

### Example

Placement Readiness Score: **72 / 100**

This helps students understand **how ready they are for placements**.

Users upload their resume and the system performs:

✔ ATS Score Calculation
✔ Keyword Detection
✔ Grammar Checking
✔ Role-Based Suggestions

Suggestions:

* REST API

# 💼 Job Recommendation System

INDIAIN fetches job listings from APIs and compares them with user skills.

### Example

Flipkart SDE Intern — 75% Match
Amazon Backend Intern — 70% Match

Each recommendation explains **why the job matches the user's profile.**

Using the GitHub API, INDIAIN verifies developer activity.

It analyzes:

* Repository count
* Languages used
* Commit activity

✔ Verified Developer
✔ Active Contributor

# 🚀 Getting Started

## 1️⃣ Clone the repository

git clone https://github.com/yourusername/INDIAIN.git

cd INDIAIN

---

## 2️⃣ Setup Backend

cd backend

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

uvicorn main:app --reload

---

## 3️⃣ Setup Frontend

cd frontend

npm install

npm run dev

Frontend will run on:

http://localhost:5173
